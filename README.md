## 🛠️ Repository Architectural Features

Rather than relying on massive component libraries, this single-page app is engineered from the ground up to showcase clean, performant, production-grade frontend patterns:

* **⚡ High-Velocity Asset Pipeline:** Fully bundled and optimized on-the-fly using the modern **Parcel Engine** compiler.
* **🎨 Modular Styles Architecture:** Layout parameters are broken down into isolated, single-responsibility files using an optimized **Sass (SCSS) partial folder structure**.
* **📡 Asynchronous Theme Orchestration:** Implements a reactive, system-aware **Real-Time Dark Mode** tracking circuit using the HTML5 `window.matchMedia` API. It dynamically synchronizes with device system profiles instantly and handles custom manual overrides via client-side `localStorage` state persistence.
* **📱 Performance & Optimization:** Fully responsive layout with custom, responsive graphic cards built to showcase my projects including hackathon solutions and tournament tracking milestones.
